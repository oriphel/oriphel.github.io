## About The Project
Los LLMs han llegado para establecerse como solucionadores de problemas, abordando desde las
cuestiones cotidianas hasta los desafíos empresariales más complejos, y extendiéndose incluso a áreas tan
diversas como la investigación científica, los servicios estatales y mucho más. Esta tecnología está liderando
una auténtica revolución tecnológica de la cual Latinoamérica no se puede quedar afuera. Esta iniciativa es
una invitación a la comunidad de IA en Latinoamérica a unir fuerzas para desarrollar un primer gran modelo
de lenguaje de código abierto que sea una semilla inicial para embarcar a nuestra región en esta revolución.
